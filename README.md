# React Boilerplate

## Technologies

* [x] React
* [x] Webpack
* [x] Babel
* [ ] Karma
* [ ] Jasmine
* [ ] PhantomJS

## Getting Started

```sh
$ npm install -g webpack webpack-dev-server karma-cli
$ npm start
```

Run the test suite

```sh
$ npm test
```

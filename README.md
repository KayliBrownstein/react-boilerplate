# React Boilerplate

## Technologies

* [x] React
* [x] Webpack
* [x] Babel
* [x] Karma
* [x] Jasmine
* [x] PhantomJS

## Getting Started

Install dependencies and start the application:

```sh
$ npm install -g webpack webpack-dev-server karma-cli
$ npm start
```

Run the test suite once:

```sh
$ npm test
```

Run the test suite, continuously:

```sh
$ karma start
```
